대응법
 처음 왔는지 묻고, 다음을 따른다

회원이 처음인 경우

1. 연습사용허가신청서 작성 부탁하기
주소(면/동 정도만),성명,전화번호만 작성 부탁 (나머지는 직원이 작성)
2. 결제 (결제방법 클릭)
3. 필요 시 채와 탁구공*을 무료대여해주고 내보낸다.
*(에어컨 버튼 밑에 있음)
4. 신청서 내용을 바탕화면의 일회원 부분에 있는 "일일회원 데이터" 엑셀 파일에 입력한다.

회원이 처음이 아닌 경우

1. 이름 확인하기
2. 일일회원데이터.xlsx 파일에서 이름 검색하기(엑셀파일의 빈 화면을 클릭하고 컨트롤 f)
이때 검색결과가 나오는지 확인한다
검색결과가 나온다면,
결제 후 회원 내보내기 (결제방법 클릭)
일일회원 파일 보면서 신청서 대필.
검색결과가 없다면,
주소(면/동 정도만),성명,전화번호만 작성 부탁 (나머지는 대신 작성)
결제 후 회원 내보내기 (결제방법 클릭)
일일회원데이터 파일 마지막 줄에 데이터 입력
